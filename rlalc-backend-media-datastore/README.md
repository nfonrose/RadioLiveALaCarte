
# RadioLiveALaCarte backend-media-datastore



## Introduction

This components, based on LanceDB and a custom Python Backend, is responsible for:
 - Storing all the media files/segments downloaded by the 'rlalc-backend-media-capture-service'
 - Storing all the metadata associated with the media and generated by other components of the platform ('rlalc-backend-media-labeler-...' components)
 - Allowing querying of the media based on the metadata
