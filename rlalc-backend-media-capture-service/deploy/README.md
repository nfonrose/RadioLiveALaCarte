
# Deployment




## Deployment for **Dev** activities

### Docker compose

This directory contains deployment artefacts (docker compositions, configuration files, ...) for the Dev environment.
 - Reference local Docker images (not the ones pushed to Github Container Registry)
 - With a `parameters.env` containing values for the Dev environment




## Deployment for **PROD** environments

### Docker compose

The Docker composition for PROD deployments are located in the `/deploy` top-level directory of the RLALC project.
